# iNeuronAi

Ineuron - Assignment 2

## pre-requiste

node js

## Building and running on localhost

To run Triangle Problem ,

```sh
npm run triangle
```

To run find Grade Problem Using Switch,

```sh
npm run grade
```

To run Multiples of 3 & 5 problem,

```sh
npm run multiples
```

To run Prime Number Problem,

```sh
npm run prime
```
